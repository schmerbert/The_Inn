| Channel | What | How it arrives |
|---------|------|----------------|
| **Breath (inhale)** | Arrival packet — posture, standing context, seam, warnings, room | Tool return on wake (Trinity coupling) |
| **Breath (exhale)** | Departure check — stale handoff, open DR, unfixed seat | Tool return or CLI before leave (Lighthouse) |
